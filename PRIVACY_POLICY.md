# PromptSmith Privacy Policy

## Data Collection
PromptSmith does not collect, store, or transmit your prompts to any server controlled by PromptSmith.

## Local Processing
All prompt classification runs locally in your browser using an on-device ONNX model. Your prompts never leave your device for classification purposes.

## API Usage
If you choose to add a Groq or Gemini API key, your prompts are sent to those services for enhancement only when the local classifier confidence is below 72%. This is optional. You control which API keys you add.

## API Keys
Keys are stored in Chrome sync storage and are never sent to PromptSmith servers because PromptSmith has no servers.

## No Analytics
PromptSmith collects no usage analytics, no telemetry, and no personal data.
