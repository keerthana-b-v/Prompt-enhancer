PromptSmith 🛠️✨ — Premium, Technique-Aware AI Prompt Enhancement

Take absolute control of your LLM prompts inside ChatGPT, Claude, and Gemini with PromptSmith! PromptSmith is a zero-latency, privacy-first, zero-cost browser extension that automatically classifies and optimizes your raw prompts in real-time using academically proven prompting techniques—offline or cascade-routed to cloud APIs.

=== 🚀 KEY FEATURES ===
* 🧠 Edge ML Classification: Instantly detects the structural intent of your prompt (e.g. math, coding, creative, factual, planning) using a highly optimized, dynamic transformer model running locally in your browser.
* ⚡ Smart "Reasoning Model" Detection: Recognizes when you are interacting with thinking models (OpenAI o1/o3-mini, Claude Thinking, DeepSeek R1) and automatically shifts from heavy structural additions to lightweight cognitive context to save tokens and prevent conflicts.
* 🔗 8 Modular Prompting Strategies: Seamlessly implements Chain-of-Thought (CoT), Program-of-Thought (PoT), Least-to-Most decomposition, Step-Back abstraction, and more.
* 📊 Token-Cost Counters: View word-to-token multipliers in real-time and toggle "Token-Efficient Mode" to compress prompts without losing context.
* 🔒 100% Privacy-First: Bypasses central servers entirely. Your prompts are analyzed locally on-device. API keys are stored securely using Chrome sync.

=== 🧠 HOW IT WORKS ===
1. Type your prompt in ChatGPT, Claude, or Gemini.
2. The local classifier pre-warms on textarea focus, running inference in under 15ms.
3. Click "Enhance" to apply the optimal mathematical formula. Use the popup settings to customize providers (Groq, Gemini, or OpenRouter) or manual overrides.

=== 📦 LOCAL MODEL SETUP ===
For optimal edge ML performance:
1. Clone the PromptSmith repository.
2. Run our 8-bit dynamic INT8 quantization notebook to compile the 15MB Edge Classifier model.
3. Place `model_quantized.onnx` inside the extension's model directory and load unpacking to enjoy zero-cost, private prompt routing!
